# AI Interview Platform

A role-based interview platform built using Python and Flask.

## Current Features

* User Registration  
* User Login / Logout  
* Database Integration  
* Role-Based Interview Questions  
* Answer Evaluation  
* Score Calculation  
* Results Page  
* Flask Backend  

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS

## Upcoming Features

* Resume Upload  
* AI Role Suggestion  
* Timer Based Interviews  
* Progress Bar  
* Random Questions  
* Anti Copy/Paste Protection  
* Admin Dashboard  
* Interview History
